# Screening-B3
Esse notebook tem como objetivo filtrar as ações da B3 de acordo com os critérios do usuário. No momento, os critérios são ações até R$ 20,00 e liquidez diária igual ou maior a 50M. Ao final do código é gerado um dataframe que filtra todas ações listadas na B3 de acordo com os critérios estabelecidos pelo usuário.
